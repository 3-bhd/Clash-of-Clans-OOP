# Clash Off Clans

Clash of Clans inspired strategy game in C++ and Qt · Dijkstra pathfinding · OOP · Data Structures

---

## Overview

A fully functional Clash of Clans inspired strategy game built in C++ with a Qt-based 2D GUI. The game features intelligent troop pathfinding using Dijkstra's algorithm, physics-based shooting mechanics, and a progressive level unlocking system. Built using core OOP principles and data structures.

Built as part of a Data Structures & OOP course at The American University in Cairo.

---

## Features

- 🗺️ **Dijkstra Pathfinding** — troops intelligently navigate the map using shortest path algorithm
- 💣 **Shooting Dynamics** — physics-based bullet and cannon mechanics
- 🔓 **Level Unlocking System** — progressive difficulty with unlockable levels
- 🎮 **Qt 2D GUI** — fully rendered graphical interface with assets and sound
- 🏗️ **OOP Architecture** — clean class hierarchy following OOP principles
- 📊 **Data Structures** — graphs, queues, and priority queues for game logic

## Gameplay

- Deploy troops that automatically find the shortest path to targets using Dijkstra's algorithm
- Cannons and bullets follow physics-based trajectories
- Complete levels to unlock progressively harder challenges

## Running the Game

1. Open the project in Qt Creator
2. Build and run using the `.pro` project file
```bash
# Or from command line
qmake ClashOffClans.pro
make
./ClashOffClans
```
